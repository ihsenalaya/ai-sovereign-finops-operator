# LaTeX source — Economic-Aware and Sovereignty-Constrained Routing for Enterprise LLM Gateways

Self-contained LaTeX project. All paths are relative; nothing outside this folder
is needed.

```
latex/
├── main.tex          # the full article (13 pages)
├── references.bib    # bibliography (BibTeX)
├── figures/          # all figures used by the paper (PNG)
└── main.pdf          # last compiled output
```

## Compile on Overleaf (recommended for sharing with a reviewer)
1. Upload `greenops-paper-overleaf.zip` to Overleaf (**New Project → Upload Project**).
2. Overleaf auto-detects `main.tex` as the main file.
3. Menu → **Compiler: pdfLaTeX**, then **Recompile**. The bibliography builds
   automatically (BibTeX). No extra packages to install.

## Compile locally
- **Tectonic** (single binary, fetches packages on first run):
  ```bash
  tectonic main.tex
  ```
- **TeX Live** (pdflatex + bibtex):
  ```bash
  pdflatex main && bibtex main && pdflatex main && pdflatex main
  ```

## Notes
- Document class is `article` (11pt, a4). Swap `\documentclass` for `IEEEtran`
  or `acmart` when the target venue is fixed.
- Numbers are traceable to the committed runs in `../../results`,
  `../../results-stats`, and `../../results-bench`; figures are generated by the
  scripts in `../../scripts`.
